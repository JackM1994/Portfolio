About

Web Developer

I like to code all my websites from scratch. I use markup languages like HTML to create friendly user websites. I start by creating mockups and prototypes to show my clients or when I am doing small projects for myself. It gives the client an understanding of how the website will appear and saves huge amount of coding time. For styling my webpages and websites, I use CSS and SASS. For interactions and animations, I use JavaScript and JQuery They make the website and webpages more exciting and just add nice visual effect.

Mobile Developer

I develop mobile applications for Android software using Android Studio. I find this program user-friendly, anything I code I can see in real time which makes it perfect for showing clients. I make protoypes and mockups for the client to see what their mobile application will look and possible, how it will run. The languages I use are Java and XML. I find these languages are perfect for developing Android Applications.

Designer

I opted for simple layouts, with very clean design and patterns. Making content easy to read and making any interaction smooth and seemless.
