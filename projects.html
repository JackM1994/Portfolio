CSS Guide

About

This project gives the user a sense of how CSS works. The single web page consists of, typography, buttons, single forms, images and grid systems. The project is also interactive as the user is able to click and select from nav bar options. These options direct the user to the great section of the page. With different types of colours, fonts and layouts, this web page, gives the user an idea of how CSS can brighten up and make any web page more aesthically pleasing to the eye.
