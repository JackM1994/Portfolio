CSS Guide

About

This project gives the user a sense of how CSS works. The single web page consists of, typography, buttons, single forms, images and grid systems. The project is also interactive as the user is able to click and select from nav bar options. These options direct the user to the great section of the page. With different types of colours, fonts and layouts, this web page, gives the user an idea of how CSS can brighten up and make any web page more aesthically pleasing to the eye.

Photo Gallery

About

This project is a simple photo gallery. The project displays 12 images in a grid form when the screen is greater than 769px. Using JQuery plugin, the user can search for specific images. Once the user clicks on an image, a slideshow screen appears. The slideshow can be manually or automatically controlled. If the user so wishes to download an image, they are given the download option in the slideshow view.

Languages used :
Gallery's Main Page
Search filter
Gallery Slideshow
